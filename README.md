# About
**amCharts** is a Javascript-based visual framework created by *Anatanas Marcelionis* from Vilnius, Lithuania.

# Products
* **Javascript Charts**
  * Serial Plots
    * Line
    * Column
    * Bar
  * Coordinate Plots
    * XY
    * Scatter
    * Bubble
  * Sliced Plots
    * Pie
    * Funnel
    * Pyramid
  * Other 
    * GANTT
    * Polar
    * Radar
    
* **JavaScript Stock Charts**
  * Comparison
  * Events
  * Horizon
  
* **JavaScript Maps**
  * Heat
  * Weather
  * Flight Route
